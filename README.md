# ☕ Projeto Cafeteria

🚀 Desenvolvi um projeto pessoal: um **site de cafeteria**, utilizando apenas 
- **HTML
- CSS**.  

> O objetivo foi praticar e consolidar meus conhecimentos em estruturação de páginas e estilização, criando uma experiência visual agradável e organizada.

## ✨ Principais pontos do projeto
- Estrutura semântica em **HTML** para melhor organização e acessibilidade.  
- Layout **responsivo em CSS**, adaptado para diferentes dispositivos.  
- Design inspirado em cafeterias modernas, com foco em estética e usabilidade.  

## 🎯 Objetivo
Este projeto representa meu empenho em evoluir como desenvolvedor front-end e minha paixão por transformar ideias em interfaces funcionais e atraentes.
